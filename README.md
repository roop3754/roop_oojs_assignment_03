# Fakebook Web Page

This is a simple social media-style web page where users can post **text messages**, **images**, or both. It demonstrates basic **JavaScript classes**, **ES modules**, and **DOM manipulation**.  

Users can see their posts displayed with their profile picture, name, and the current date and time. A pop-up modal shows account information using the `Subscriber` class.

---

## Features

- Post **text**, **images**, or both.  
- Posts display profile picture, user name, and timestamp.  
- Account info modal shows user details like pages, groups, and monetization ability.  
- Implemented using **User** and **Subscriber** classes with private properties and `getInfo()` methods.
